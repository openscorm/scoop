# Scoop

Scoop is an affordable, open-source alternative to SCORM Cloud for managing and delivering training content.

## About

Scoop provides flexible solutions for organizations that need to deliver SCORM content:

- **Self-hosted**: Deploy on your own infrastructure for complete control over your learning content and data
- **Managed and hosted**: Let us handle the infrastructure and maintenance while you focus on your content

Whether you have technical expertise and want full control, or prefer a hassle-free managed solution, Scoop has you covered. Built with modern web technologies, it offers everything you need to  upload and host your SCORM packages, for content delivery in any Learning Management System.

## Status

This project is in early development. We are actively building out the core features and documentation. Content and code examples will be added regularly as development progresses.

## Planned Features

- SCORM 1.2 support
- xAPI (Tin Can API) support
- RESTful API
- SCORM package upload and configuration
- SCORM training content delivery (with progress tracking)
